# AI Incident Severity Classifier

This project uses Large Language Models to classify incident severity automatically.

## Purpose:
Reduce manual triage effort in L2 support.

## Features:
• Automatic severity classification
• Business impact analysis
• Recommended actions

## Tech:
• Prompt Engineering
• LLM classification
• GitHub version control

## Real-World Use Case

This classifier simulates automated triage used in ITSM tools
to reduce manual L2 severity determination time.

Applicable to:
• ServiceNow ticket triage
• Incident automation pipelines
• AI-powered support copilots

## Architecture

Incident Text-->LLM Prompt-->Severity Classification-->Business Impact + Action Recommendation
